## 介绍

掌上大交小程序，提供教务信息、图书查询、后勤报修、就业信息等服务，更有校友圈等新特性你体验。

## 更新日志

##### V1.0 

2019.12.05，完成所有功能

##### V2.0

2020.06.28，完成第二版开发，优化了教务使用体验，新增了社交功能。

## 后端

https://github.com/duniqb/mobile-dj

后端使用 Java 语言开发，技术栈：

- 开发框架：`SpringBoot2.2`
- 数据库：`MySQL8.0`
- 持久层框架：`MyBatis3.2`
- 缓存中间件：`Redis`
- 爬虫框架：`OkHttp3`，`Jsoup`
